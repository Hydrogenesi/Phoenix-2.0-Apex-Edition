# Substrate Laws

The **Substrate Layer** contains the foundational laws that govern the fundamental mechanics of the Phoenix–Hydrogenesi Codex. These laws form the base upon which Universal and Apex laws are built.

## The Five Substrate Laws

1. **[Conservation](./Conservation.md)** — Substrate Continuity
2. **[Symmetry](./Symmetry.md)** — Mirror‑Operator Invariant
3. **[Recursion](./Recursion.md)** — Scale‑Repetition Engine
4. **[Emergence](./Emergence.md)** — Complexity Genesis
5. **[Duality](./Duality.md)** — Fundamental Polarity

## Structure

Each substrate law document contains:

- **Statement** — The core principle
- **Meaning** — Detailed explanation with bullet points
- **Sigil** — Visual representation in ASCII art
- **Cross‑References** — Links to corresponding Universal and Apex laws
