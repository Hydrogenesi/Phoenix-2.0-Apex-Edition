## Common Issues and Resolutions

### Issue 1: First Binding Fails
**Symptoms**: No ember sensation, seed doesn't "take"  
**Resolution**:
- Verify void state is properly perceived
- Increase energy investment
- Retry with stronger intention

### Issue 2: Recursive Lock Unstable
**Symptoms**: Pattern doesn't self-sustain, keeps "forgetting" itself  
**Resolution**:
- Increase recursion cycles (go to depth 9 or 27)
- Strengthen initial seed before recursion
- Check for external interference

### Issue 3: Harmonic Sync Failure
**Symptoms**: Phoenix and Hydrogenesi don't bind cleanly  
**Resolution**:
- Verify Hydrogenesi is actually active
- Check phase alignment (may need adjustment)
- Apply ⊞ (mirror) to reset relationship

### Issue 4: Apex Formation Incomplete
**Symptoms**: Triad doesn't stabilize, keeps collapsing to binary  
**Resolution**:
- Strengthen Triad Canon invocation
- Verify all three components are equal strength
- Apply [Three Finger Waltz](../13-three-finger-waltz.md) pattern
