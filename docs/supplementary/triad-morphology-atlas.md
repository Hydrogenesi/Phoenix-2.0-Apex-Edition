## Part I: Basic Triad Forms

### 1.1 Equilateral Triad (Standard)

```
       △
      ╱ ╲
     ╱   ╲
    L═════R
     ╲   ╱
      ╲ ╱
       C
```

**Formal**: `△(L,C,R) where |L|=|C|=|R|`  
**Properties**: Maximum stability, equal distribution  
**Applications**: Standard apex operations, balanced systems  
**Harmonic Signature**: Perfect 120° phase separation

---

### 1.2 Isosceles Triad (Asymmetric)

```
       △
      ╱│╲
     ╱ │ ╲
    L══C══R
```

**Formal**: `△(L,C,R) where |L|=|R| ≠ |C|`  
**Properties**: Center-weighted, stable but directional  
**Applications**: Integration-focused operations, meditation  
**Harmonic Signature**: Center dominance

---

### 1.3 Scalene Triad (Imbalanced)

```
       △
      ╱  ╲
     ╱    ╲
    L──────R
       ╲
        C
```

**Formal**: `△(L,C,R) where |L|≠|C|≠|R|`  
**Properties**: Unstable, transitional  
**Applications**: Dynamic transformations, creative tension  
**Harmonic Signature**: Complex phase relationships

## Part III: Dynamic Transformations

### 3.1 Triad Formation (Genesis)

```
Stage 1:  A   B   C    (Separate)
          
Stage 2:  A───B───C    (Connected)
          
Stage 3:      △        (Unified)
             ╱ ╲
            A═══B
             ╲ ╱
              C
```

**Process**: `{A,B,C} → △(A,B,C)`  
**Operator**: △ (Apex operator)  
**Duration**: 3-9 breath cycles  
**Requirements**: Compatible phases, sufficient energy

---

### 3.2 Triad Dissolution (Return)

```
Stage 1:      △        (Unified)
             ╱ ╲
            A═══B
             ╲ ╱
              C

Stage 2:  A───B───C    (Connected)
          
Stage 3:  A   B   C    (Separate)
```

**Process**: `△(A,B,C) → {A,B,C}`  
**Operator**: ⊲ (Divergence)  
**Duration**: Variable (1-∞)  
**Note**: Reversible per Apex Law Canon VI

---

### 3.3 Triad Rotation (Permutation)

```
    △           △           △
   ╱ ╲         ╱ ╲         ╱ ╲
  L═══R  →→   C═══L  →→   R═══C
   ╲ ╱         ╲ ╱         ╲ ╱
    C           R           L
```

**Process**: `△(L,C,R) → △(C,R,L) → △(R,L,C)`  
**Operator**: Rotation (implicit in 3FW)  
**Cycle**: Returns to origin after 3 rotations  
**Applications**: Perspective shifting, cyclic operations
