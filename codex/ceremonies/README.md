## Usage Guidelines

### Before Any Ceremony
1. **Read the complete documentation** for the ceremony
2. **Verify all prerequisites** are met
3. **Ensure crown authority** is granted
4. **Understand the consequences** of the ceremony

### If a Ceremony is Interrupted
1. Stop all running processes
2. Run cleanup commands if provided
3. Document partial state
4. Restart from beginning when ready

1. Execute emergency deactivation if available
2. Review logs for root cause
3. Restore from backup if necessary
4. Do not attempt ceremony again until stable
