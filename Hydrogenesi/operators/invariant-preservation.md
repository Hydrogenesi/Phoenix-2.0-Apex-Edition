## Formal Definition

```
P: Ψ → P(Ψ)

where:
  Ψ = Pattern state
  P(Ψ) = Set of invariant properties that must be preserved
```

### Domain
- **Source**: Pattern state in transformation space
- **Target**: Set of essential invariants that survive all transformations
- **Topology**: Invariant subspace within pattern space

### Invariants
1. **Persistence**: Properties in P(Ψ) cannot be destroyed
2. **Heredity**: P(Op(Ψ)) ⊇ P(Ψ) for structure-preserving operators
3. **Essentiality**: Only truly fundamental properties are included

## The Preservation Theorem

### Theorem: Invariant Monotonicity
```
For any pattern Ψ and structure-preserving operator Op:
  P(Op(Ψ)) ⊇ P(Ψ)

Proof:
  Assume Ψ has invariant set P(Ψ) = {p₁, p₂, ..., pₙ}
  
  Op is structure-preserving, so:
    For each pᵢ ∈ P(Ψ):
      pᵢ survives transformation
      pᵢ ∈ P(Op(Ψ))
  
  Op may introduce new invariants pₙ₊₁, ...
  
  Therefore: P(Op(Ψ)) ⊇ P(Ψ) ∎

Exception: Void operator ⊝ destroys all invariants.
```
