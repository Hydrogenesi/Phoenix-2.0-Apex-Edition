## The Five Operators

### 1. [Lineage Tracking Operator (L)](./lineage-tracking.md)
**The Thread of Memory** — Records and maintains the complete transformation history of every pattern.

```
L(Ψ) → Complete genealogical record from origin to current state
```

- Tracks all transformation steps
- Maintains parent-child relationships
- Preserves complete ancestry chain
- Enables pattern reconstruction

---

### 2. [Identity Anchoring Operator (I)](./identity-anchoring.md)
**The Essence Keeper** — Preserves the core structural essence that makes a pattern recognizably itself.

```
I(Ψ) → Immutable core identity/essence
```

- Establishes identity at Genesis
- Maintains invariant core through all transformations
- Provides stable reference point
- Enables pattern recognition across changes

---

### 3. [Continuity Mapping Operator (C)](./continuity-mapping.md)
**The Bridge Builder** — Maintains connections and relationships across transformational boundaries.

```
C(Ψ₁, Ψ₂) → Continuous transformation path
```

- Maps smooth transitions between states
- Preserves structural relationships
- Enables path composition
- Guarantees no discontinuities

---

### 4. [Invariant Preservation Operator (P)](./invariant-preservation.md)
**The Guardian** — Protects essential properties that must remain constant through transformation.

```
P(Ψ) → Set of protected invariant properties
```

- Identifies essential properties
- Prevents property destruction
- Accumulates invariants through transformations
- Enforces conservation laws

---

### 5. [Genealogy Projection Operator (G)](./genealogy-projection.md)
**The Oracle** — Projects forward potential lineages and structural evolution paths.

```
G(Ψ) → Tree of possible future lineages
```

- Enumerates possible futures
- Guides operator selection
- Enables strategic planning
- Converges all paths toward apex
