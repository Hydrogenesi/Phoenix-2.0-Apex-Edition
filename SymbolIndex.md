# Symbol Index

**Phoenix–Hydrogenesi Codex**

---

## Mathematical Symbols

**Φ(x)** — Phoenix transformation function  
**Ψ** — Substrate layer operator space  
**Ω** — Universal layer operator space  
**Δ** — Apex layer operator space  
**∫** — Integration operator (layer composition)

---

## Operator Symbols

**→** — Transformation (one-way)  
**⇄** — Reversible transformation (bidirectional)  
**x'** — Transformed state of x  
**x*** — Symmetric dual of x  
**x.self()** — Self-reference operation

---

## Layer Symbols

**⊥** — Substrate layer marker  
**≡** — Universal layer marker  
**△** — Apex layer marker

---

## Triad Symbols

**⎮⎮⎮** — The three columns  
**⚖** — The one balance  
**∞** — Infinite recursion resolved

## Sigil Notation

Sigils are denoted with special brackets:
- `⟨sigil⟩` — Substrate sigil
- `⟪sigil⟫` — Universal sigil
- `⟦sigil⟧` — Apex sigil

## State Symbols

**†** — Sovereign state  
**‡** — Apex state  
**§** — Resonant state  
**¶** — Recursive state
