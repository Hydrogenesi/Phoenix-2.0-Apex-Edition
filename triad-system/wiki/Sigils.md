# Sigils

**Sigils** are the visual symbolic representations of the eight Phoenix 2.0 operators and key Triad System concepts.

## Overview

Each operator has a unique sigil that serves as:
- **Visual identifier**: Quick recognition
- **Symbolic meaning**: Encoded semantics
- **Ceremonial marker**: Ritual invocation symbol
- **Documentation aid**: Clear operator reference

## Sigil Usage

### In Documentation

Use sigils to mark operator usage:

```
⊕ Genesis(seed) → initial_pattern
⊗ Harmonic(initial_pattern) → stable_pattern
```

### In Rituals

Mark ritual phases with sigils:

```
Phase 1: ⊝ Void invocation
Phase 2: ⊕ Genesis
Phase 3: ⊗ Harmonic stabilization
Phase 4: △ Apex sealing
```

### In Diagrams

Use sigils in system diagrams:

```
[⊕] → [⊗] → [⊛] → [△]
 ↑                    ↓
[⊝] ← ← ← ← ← ← ← ← ← 
```

### In Code Comments

Mark operations in code:

```python
# ⊕ Genesis: Initialize new kernel
kernel = initialize_kernel(seed)

# ⊗ Harmonic: Stabilize
kernel = harmonize(kernel)

# △ Apex: Seal
kernel = seal_kernel(kernel)
```

## Sigil Combinations

### Operator Sequences

Chain sigils to represent sequences:

```
⊕ → ⊗ → ⊛ → △  (Complete transformation)
⊕ → ⊗ → ⊝      (Create and dissolve)
⊳ → ⊗ → △      (Converge to apex)
```

### Parallel Operations

```
⊕ ∥ ⊕  (Parallel genesis)
⊗ ∥ ⊗  (Parallel harmonization)
```

### Conditional Operations

```
⊕ → (⊗ | ⊝)  (Harmonize or void)
```

## Sigil Properties

### Mathematical Properties

Each sigil represents an operator with properties:

**Associative:**
```
(⊕ ⊕ ⊕) A = ⊕ (⊕ ⊕ A)
```

**Commutative (some):**
```
⊗ A ⊗ B = ⊗ B ⊗ A
```

**Idempotent (some):**
```
△ △ A = △ A
```

### Visual Properties

**Symmetry:**
- Genesis (⊕): 4-fold rotational symmetry
- Harmonic (⊗): 4-fold rotational symmetry  
- Apex (△): 3-fold rotational symmetry
- Void (⊝): 2-fold rotational symmetry

**Complexity:**
- Simple: ⊕, ⊝, △
- Moderate: ⊗, ⊞
- Complex: ⊛, ⊳, ⊲

## Ceremonial Usage

### Invocation Gesture

Each sigil can be traced as a gesture:

**Genesis (⊕):**
1. Draw circle clockwise
2. Draw vertical line (down-up)
3. Draw horizontal line (left-right)

**Apex (△):**
1. Draw left edge (bottom-up)
2. Draw right edge (top-down)
3. Draw base (left-right)

**Void (⊝):**
1. Draw circle clockwise
2. Draw horizontal line (left-right)
