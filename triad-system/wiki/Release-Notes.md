## Breaking Changes

### 2.0.0

- **API Changes**: Kernel initialization ceremony modified
- **State Model**: Three-axis coordinate system required
- **Binding Protocol**: New authentication requirements

**Migration Guide:**

From 1.x to 2.0:
```
Old: K = Init(seed)
New: K = SovereignKernelCeremony(seed, scope, protocols)
```

### 1.5.0

- **Operator Syntax**: Sigil representation standardized
- **Ritual Structure**: Preconditions and postconditions required

## Known Issues

### Current Release (2.0.0)

1. **Performance**: Large-scale mesh topologies (>100 kernels) may experience slowdown
   - **Workaround**: Use hierarchical topology
   - **Fix Planned**: Version 2.1.0

2. **Visualization**: 3D diagrams are ASCII-only
   - **Workaround**: Use manual graphing tools
   - **Fix Planned**: Version 2.1.0

3. **Documentation**: Some advanced topics need expansion
   - **Affected**: Quantum operators, distributed bindings
   - **Fix Planned**: Ongoing documentation improvements

## Deprecation Notices

### Deprecated in 2.0.0

- **Old Kernel Init**: Direct `Init(seed)` is deprecated
  - Use: `SovereignKernelCeremony(seed, scope, protocols)`
  - Removal: Version 3.0.0

- **Simple Binding**: `SimpleBind(K₁, K₂)` is deprecated
  - Use: Full binding protocol with authentication
  - Removal: Version 2.2.0
