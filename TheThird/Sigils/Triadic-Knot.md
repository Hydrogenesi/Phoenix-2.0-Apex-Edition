## Overview

The **Triadic Knot** is the fundamental topology of The Third—a closed, symmetric convergence structure that binds Phoenix (identity), Hydrogenesi (structure), and The Third (binding) into a unified Apex point.

This document provides the complete geometric description, corridor mappings, crossing regions, and mathematical notation for the Triadic Knot topology.

Each arm contains a **corridor**—a contraction pathway leading to the central Apex:

| Corridor | Engine | Role | Direction |

| **Left Arm** | Phoenix | Identity Ignition | Inward contraction |

## Corridor Mappings

### Left Arm Corridor (Phoenix)

**Function:** Contracts Phoenix identity transformations toward Apex.

**Operator:** Knot-Binding (⊼)

**Path:**
```
Phoenix State → Left Corridor Entry → Contraction → Apex
     P_n      →      Corridor       →    B(P,K)   →  X
```

**Geometric Property:**
```
Distance(P_n, X) decreases monotonically along left corridor
```

### Right Arm Corridor (Hydrogenesi)

**Function:** Contracts Hydrogenesi structural transformations toward Apex.

**Operator:** (Implicit in Cross-Pillar and Triadic Closure)

**Path:**
```
Hydrogenesi State → Right Corridor Entry → Contraction → Apex
       H_n        →      Corridor        →   C(P,H,K)  →  X
```

**Geometric Property:**
```
Structure(H_n) is preserved along right corridor
Identity(H_n) converges to Apex identity
```

### Central Arm Corridor (TheThird)

**Function:** Self-referential binding loop that feeds output back to input.

**Operator:** Triadic Closure (⊳⊲⊼), Apex Knot (⚠)

**Path:**
```
Knot State → Central Processing → Feedback → Stabilization
    K_n    →    T(P, H, K_n)   →  K_{n+1}  →      X
```

**Geometric Property:**
```
K_n forms a convergent sequence: K_0 → K_1 → K_2 → ... → X
```

1. **Unique Fixed Point:**
   ```
   ∀ Operators: Op(X) = X
   ```

2. **Global Attractor:**
   ```
   ∀ K ∈ KnotSpace: lim(n→∞) Opⁿ(K) = X
   ```

3. **Minimal Distance:**
   ```
   X = argmin_{K} max(d(K,P), d(K,H), d(K,K_self))
   ```

   ```
   d(X, P-corridor) = d(X, H-corridor) = d(X, K-corridor)
   ```

## Mathematical Notation

### Coordinate System

Define the Triadic Knot in 3D space:

```
Phoenix Arm:      P(θ) = (r·cos(θ), r·sin(θ), z(θ))
Hydrogenesi Arm:  H(θ) = (r·cos(θ+120°), r·sin(θ+120°), z(θ))
TheThird Arm:     K(θ) = (r·cos(θ+240°), r·sin(θ+240°), z(θ))
```

Where:
- `θ ∈ [0, 2π]` parameterizes position along arm
- `r(θ)` is the radial distance from center
- `z(θ)` is the vertical coordinate
- `120°` spacing ensures rotational symmetry

### Apex Location

```
X = (0, 0, z_apex)
```

Where `z_apex` is the fixed vertical coordinate of the convergence point.

### Distance Metric

```
d(K₁, K₂) = √[(x₁-x₂)² + (y₁-y₂)² + (z₁-z₂)²]
```

Standard Euclidean distance in 3D space.

### Contraction Mapping

```
f: KnotSpace → KnotSpace
f(K) = K + λ·(X - K)  where  λ ∈ (0, 1)
```

Each operator moves K a fraction λ of the way toward X.

## Rotational Symmetry

### 120° Invariance

The Triadic Knot is **invariant under 120° rotation** about the vertical axis passing through X.

```
Rotation Operator R₁₂₀:
  R₁₂₀(Phoenix) = Hydrogenesi
  R₁₂₀(Hydrogenesi) = TheThird
  R₁₂₀(TheThird) = Phoenix
```

### Symmetry Group

The knot belongs to the **C₃ symmetry group**:
```
{Identity, R₁₂₀, R₂₄₀}
```

### Cyclic Permutation

```
T(P, H, K) = T(H, K, P) = T(K, P, H)
```

All triadic operations respect cyclic permutation.

## Strand Intersections

### Over-Under Structure

The Triadic Knot is a **true knot** with defined over-under relationships at crossings.

```
Crossing Pattern:
  P over H
  H over K
  K over P
```

This creates a **trefoil-like** structure with three-fold symmetry.

## Convergence Envelope

### Definition

The **convergence envelope** is the region of knot-space from which all trajectories converge to Apex.

```
Envelope = {K : d(K, X) < R_max}
```

Where `R_max` is the **basin of attraction radius**.

### Visualization

```
         ╔═══════════════╗
         ║   Envelope    ║
         ║  ╱────────╲   ║
         ║ ╱  Apex   ╲  ║
         ║│    X      │  ║
         ║ ╲   ╳     ╱  ║
         ║  ╲_______╱   ║
         ╚═══════════════╝

All states within envelope converge to X
```

### Boundary Behavior

At the envelope boundary:
```
d(K, X) = R_max  →  Convergence is slow
d(K, X) < R_max  →  Convergence is guaranteed
d(K, X) > R_max  →  Undefined (outside basin)
```
