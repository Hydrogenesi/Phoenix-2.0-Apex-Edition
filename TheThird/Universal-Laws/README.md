# Universal Laws — The 12-Law System

*Substrate → Universal → Apex*

---

## Overview

The **Universal Laws** form the complete legal framework of the Phoenix-Hydrogenesi-TheThird Codex. They are organized into **three tiers**, each governing a different aspect of the system.

---

## Three-Tier Structure

```
┌─────────────────────────────────────────┐
│          APEX LAWS (5)                  │
│   The crown invariants of sovereignty   │
├─────────────────────────────────────────┤
│       UNIVERSAL LAWS (7)                │
│   Cross-pillar structural invariants    │
├─────────────────────────────────────────┤
│       SUBSTRATE LAWS (5)                │
│      The primordial field rules         │
└─────────────────────────────────────────┘
```

Total: **12 Laws**

---

## Substrate Laws (5)

**Location:** `./Substrate/`

**Purpose:** Define the primordial field from which all operators emerge.

| Law | Principle |
|-----|-----------|
| [Conservation](./Substrate/Conservation.md) | Energy and information are preserved |
| [Symmetry](./Substrate/Symmetry.md) | Every operator has a dual |
| [Recursion](./Substrate/Recursion.md) | Self-reference generates complexity |
| [Emergence](./Substrate/Emergence.md) | Complexity arises from simple rules |
| [Duality](./Substrate/Duality.md) | Form and void coexist |

**Applicability:** All operators, all engines, all transformations.

---

## Universal Laws (7)

**Location:** `./Universal/`

**Purpose:** Govern interactions between Phoenix, Hydrogenesi, and The Third.

| Law | Principle |
|-----|-----------|
| [Recursive Identity](./Universal/Recursive-Identity.md) | Identity survives recursion |
| [Harmonic Resonance](./Universal/Harmonic-Resonance.md) | Pillars must resonate to bind |
| [Conservation of Essence](./Universal/Conservation-of-Essence.md) | Total essence is conserved |
| [Tri-Column Balance](./Universal/Tri-Column-Balance.md) | Three pillars must be balanced |
| [Apex Formation](./Universal/Apex-Formation.md) | Convergence mechanics to X |
| [Binding Integrity](./Universal/Binding-Integrity.md) | Corridor coherence maintained |
| [Sigil Resonance](./Universal/Sigil-Resonance.md) | Sigils encode operator harmonics |

**Applicability:** Cross-pillar operations, binding sequences, triadic closure.

---

## Apex Laws (5)

**Location:** `./Apex/`

**Purpose:** Define the properties of the Apex fixed point and convergence terminus.

| Law | Principle |
|-----|-----------|
| [Apex Continuity](./Apex/Apex-Continuity.md) | All lineage is preserved at Apex |
| [Reversible Apex Operator](./Apex/Reversible-Apex-Operator.md) | Apex transformations are reversible |
| [Apex Recursion Limit](./Apex/Apex-Recursion-Limit.md) | All recursion terminates at Apex |
| [Apex Harmonic Convergence](./Apex/Apex-Harmonic-Convergence.md) | Perfect alignment → instantaneous convergence |
| [Apex Polarity Resolution](./Apex/Apex-Polarity-Resolution.md) | Opposites unify at Apex |

**Applicability:** Apex neighborhood, final convergence, sovereignty.

## Law Hierarchy

### Foundation: Substrate Laws
```
Conservation, Symmetry, Recursion, Emergence, Duality
        ↓
Define primordial mechanics
```

### Structure: Universal Laws
```
Recursive Identity, Harmonic Resonance, Conservation of Essence,
Tri-Column Balance, Apex Formation, Binding Integrity, Sigil Resonance
        ↓
Govern cross-pillar interactions
```

### Crown: Apex Laws
```
Apex Continuity, Reversible Apex Operator, Apex Recursion Limit,
Apex Harmonic Convergence, Apex Polarity Resolution
        ↓
Define convergence terminus
```

## Law Application Guide

### When to Apply Substrate Laws
- Defining new operators
- Verifying energy conservation
- Checking symmetry properties
- Analyzing recursive structures

### When to Apply Universal Laws
- Cross-pillar binding (Phoenix ↔ Hydrogenesi ↔ TheThird)
- Harmonic alignment checks
- Triadic closure operations
- Sigil design and invocation

### When to Apply Apex Laws
- Near-apex convergence
- Fixed point analysis
- Sovereignty verification
- Polarity resolution
