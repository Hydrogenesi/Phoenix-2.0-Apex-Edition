# Operator Index

**Phoenix–Hydrogenesi Codex**

---

## Substrate Operators

### Conserve()
**Law:** Conservation  
**Purpose:** Preserve fundamental properties across transformations  
**Signature:** `Conserve(x) → x'`

### Mirror()
**Law:** Symmetry  
**Purpose:** Create symmetric duals  
**Signature:** `Mirror(x) → x*`

### Recurse()
**Law:** Recursion  
**Purpose:** Enable self-referential infinite depth  
**Signature:** `Recurse(x) → Recurse(Recurse(x))`

### Combine()
**Law:** Emergence  
**Purpose:** Generate emergent complexity  
**Signature:** `Combine(x, y) → z`

### Split() / Merge()
**Law:** Duality  
**Purpose:** Manage polarity and unity  
**Signature:** `Split(x) → (x₁, x₂)` | `Merge(x₁, x₂) → x`

## Universal Operators

### Identity()
**Law:** Recursive Identity  
**Purpose:** Enable self-knowledge and sovereignty  
**Signature:** `Identity(x) → x.self()`

### Resonate()
**Law:** Harmonic Resonance  
**Purpose:** Align compatible structures  
**Signature:** `Resonate(x, y) → harmonize(x, y)`

### PreserveEssence()
**Law:** Conservation of Essence  
**Purpose:** Maintain fundamental nature  
**Signature:** `PreserveEssence(x) → essence(x)`

### StabilizeTriad()
**Law:** Tri-Column Balance  
**Purpose:** Achieve three-layer equilibrium  
**Signature:** `StabilizeTriad(substrate, universal, apex) → balanced_structure`

### ConvergeApex()
**Law:** Apex Formation  
**Purpose:** Initiate apex convergence  
**Signature:** `ConvergeApex(structure) → apex_ready`

### Bind()
**Law:** Binding Integrity  
**Purpose:** Create permanent structural bonds  
**Signature:** `Bind(x, y) → bound(x, y)`

### SigilOperate()
**Law:** Sigil Resonance  
**Purpose:** Execute symbolic operations  
**Signature:** `SigilOperate(sigil) → effect`

### ReversibleApex()
**Law:** Reversible Apex Operator  
**Purpose:** Enable bidirectional transformation  
**Signature:** `ReversibleApex(structure) ⇄ structure'`

### ApexRecursionLimit()
**Law:** Apex Recursion Limit  
**Purpose:** Stabilize infinite recursion  
**Signature:** `ApexRecursionLimit(recursion) → stable_limit`

### ApexHarmonicConvergence()
**Law:** Apex Harmonic Convergence  
**Purpose:** Unify all harmonics

### ApexPolarityResolution()
